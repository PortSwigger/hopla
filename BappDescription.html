HopLa enhances Burp Suite with keyboard-driven payload autocompletion and a comprehensive built-in payload library to streamline intrusion testing. The extension integrates with native Burp AI to provide interactive chat and quick transformation features. Custom payloads can be added via YAML configuration files to tailor the extension to specific testing requirements.

Features

  • Interactive AI chat powered by Burp AI for guidance and troubleshooting during testing
  • Quick actions to transform HTTP requests using AI instructions (convert to multipart, JSON, extract headers, etc.)
  • Built-in payload library with one-click insertion and keyboard shortcuts for common payloads
  • Search and replace functionality in Repeater for bulk modifications
  • Dynamic Burp Collaborator domain insertion via keyboard shortcut
  • Custom keyword management to add test-specific autocompletion terms on the fly
  • Copy formatted requests and responses for streamlined reporting

Usage

  1. Access the payload menu with Ctrl+Q and select from built-in payloads or add custom YAML payload files via the top menu
  2. Use keyboard shortcuts: Ctrl+J for AI Chat, Ctrl+Alt+O for Quick Actions, Ctrl+L for Search & Replace, Ctrl+M to insert Collaborator payloads
  3. Add custom keywords using Ctrl+Alt+J to extend autocompletion vocabulary during testing
